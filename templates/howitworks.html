Bird Watch: How it Works

Bird Watch is a deep learning computer vision application, developed using Keras and TensorFlow, with Flask for the web application.

What is Deep Learning?

Deep Learning is a subset of Machine Learning which focuses on an area of algorithms inspired by our understanding of how the brain works to obtain knowledge.

Read more about Deep Learning here: Codes of Interest: What is Deep Learning?



The InceptionV3 deep learning model was used as the base model and was trained using transfer learning and fine-tuning techniques.

The images are processed in 400x400 pixel resolution. The model was trained for 100 iterations on bottleneck features and then fine-tuned for a further 100 iterations.
